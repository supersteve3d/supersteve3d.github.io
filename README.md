## Site Structure

- `index.html` is the main gallery page.
- `projects/` contains one folder per project page.
- `assets/thumbnails/` contains square thumbnails used by the gallery cards.
- Shared Three.js example assets currently live in `build/`, `jsm/`, `models/`, and `textures/`.

To add a new project, create a new folder under `projects/`, add a square thumbnail under `assets/thumbnails/`, and add a matching card to `index.html`.
